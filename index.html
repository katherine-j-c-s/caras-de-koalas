¡Crea tu propio Koala!

Haz clic en cualquier parte del koala para seleccionarla y modificarla. Utiliza los botones de abajo para cambiar colores, tamaños, posiciones y añadir características.

Instrucciones:

  • Haz clic en cualquier parte del koala para seleccionarla
  • Utiliza los botones de acción para modificar la parte seleccionada:
  • Mover: Permite arrastrar el elemento con el ratón
  • Tamaño: Cambia el tamaño general del elemento
  • Color: Modifica el color de la parte seleccionada
  • Largo/Ancho: Ajusta solo el alto o el ancho del elemento
  • Girar: Rota el elemento seleccionado
  • Eliminar: Quita el elemento (podrás restaurarlo después)
  • Agregar: Añade cejas, boca u otros elementos eliminados
  • Reset: Vuelve a la versión original del koala
  • Terminado: Finaliza la edición y muestra tu koala terminado
